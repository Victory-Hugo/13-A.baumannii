# Kaptive
Kaptive是一个用于细菌荚膜多糖基因组分析的工具，该工具基于参考基因组和已知的荚膜多糖基因组信息，通过比对和注释，提供荚膜多糖基因的预测和功能注释。

# 安装
建议使用`conda`安装：
```sh
conda install kaptive
```
